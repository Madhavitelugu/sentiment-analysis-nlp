# Sentiment Analysis using NLP

##  Project Overview
This project classifies text into Positive or Negative sentiment using NLP techniques.

##  Technologies Used
- Python
- NLP
- Word2Vec
- Logistic Regression
- Scikit-learn

##  Features
- Text preprocessing
- WordCloud visualization
- Word2Vec embeddings
- Sentiment prediction model

## Model Performance
Achieved good accuracy using Logistic Regression.

##  Example
Input: "This movie was amazing"
Output: Positive
